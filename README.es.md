# Frontend Mentor - Solución Landing Page Loopstudios

Esta es mi solución al reto **Loopstudios Landing Page** de Frontend Mentor. Este proyecto se enfoca en construir una landing page completamente responsive con una estructura de layout limpia, estados interactivos en la interfaz y un menú de navegación móvil utilizando HTML, CSS y JavaScript vanilla.

El reto fue una excelente oportunidad para practicar HTML semántico, arquitectura moderna de CSS, técnicas de diseño responsive y manipulación del DOM sin depender de frameworks o librerías externas.

## Descripción general
Este proyecto es una landing page de marketing responsive para una empresa ficticia de realidad virtual llamada Loopstudios. Se adapta perfectamente a dispositivos móviles, tablets y escritorio, proporcionando un layout óptimo para cada tamaño de pantalla.

La página incluye una sección hero con efectos de overlay, un menú de navegación móvil interactivo, una sección destacada de contenido, una cuadrícula responsive de creaciones y un footer estructurado con enlaces de navegación y redes sociales.

Todo el estilo del layout se maneja con técnicas modernas de CSS, y la interacción del menú móvil está implementada con JavaScript vanilla.

---

## El reto
Los usuarios deben poder:

- Ver el layout óptimo según el tamaño de pantalla de su dispositivo.
- Experimentar un diseño responsive con enfoque mobile-first.
- Abrir y cerrar el menú de navegación móvil.
- Ver estados hover y focus en todos los elementos interactivos.
- Navegar el contenido usando el enfoque por teclado.
- Visualizar diferentes layouts para versiones móvil y escritorio.
- Ver transiciones visuales suaves y efectos de overlay.
- Experimentar una jerarquía tipográfica clara y espaciado estructurado.

## Mi proceso
- Estructuré el layout utilizando elementos **HTML5 semánticos** como `header`, `nav`, `main`, `section`, `article` y `footer`.
- Seguí un enfoque **mobile-first**, mejorando progresivamente el diseño mediante media queries.
- Construí los layouts principalmente con **Flexbox** para alineación, espaciado y responsividad.
- Utilicé **CSS custom properties (variables)** para crear un pequeño sistema de diseño para colores.
- Apliqué la metodología **BEM** para mantener el CSS modular, escalable y legible.
- Creé efectos de overlay usando **pseudo-elementos (`::before`)** para el hero y las tarjetas.
- Implementé estados hover y focus para mejorar la usabilidad y accesibilidad.
- Utilicé **position y z-index** para manejar capas y superposiciones.
- Desarrollé el toggle del menú móvil usando **JavaScript vanilla**.
- Gestioné estados de la interfaz agregando y removiendo clases dinámicamente.
- Organicé la estructura del proyecto de manera limpia para facilitar el mantenimiento.

## Lo que aprendí
- Cómo estructurar una landing page real utilizando **HTML semántico**.
- Construir layouts responsive con **Flexbox** y media queries progresivas.
- Mantener consistencia visual usando **CSS variables**.
- Crear efectos interactivos con **pseudo-elementos y transiciones**.
- Manejar capas y overlays usando `position` y `z-index`.
- Implementar un menú de navegación móvil mediante **manipulación del DOM en JavaScript**.
- Separar presentación (CSS) de comportamiento (JavaScript).
- Mejorar la experiencia de usuario con estados hover, focus y transiciones suaves.
- Organizar proyectos frontend pequeños de forma limpia y profesional.
